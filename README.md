## Setting
- Set virtual environment with conda:
> \>\> conda create -n Lab01 python=3.10.15 <br>
> \>\> conda activate Lab01
- Install necessary library:
> \>\> pip install -r requirements.txt
- Run application:
> \>\> python main.py
